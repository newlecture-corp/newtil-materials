Adaptive Layout 테스트

이 페이지를 리사이즈하면 Navigation이 자동 전환됩니다.
Compact (< 600px): 하단 Navigation Bar  |  Medium (600–839px): 좌측 Navigation Rail  |  Expanded (≥ 840px): 좌측 Navigation Drawer

카드 컴포넌트 예시

Elevated 카드

기본 그림자 카드

레이아웃 안에서 카드 컴포넌트가 정상 작동하는지 확인합니다.

Filled 카드

배경색 카드

Medium 이상에서는 좌측 Navigation이 보이고 카드가 오른쪽에 배치됩니다.

Outlined 카드

테두리 카드

Compact에서는 하단 Navigation Bar가 보이며 전체 너비를 사용합니다.

리스트 예시

  • 사용자 프로필 계정 정보를 관리합니다
  • 알림 설정 알림 기본 설정을 변경합니다
  • 보안 비밀번호 및 인증 설정

사용법 (소스 코드)

<div class="m3-layout">

  <!-- Navigation Drawer (desktop ≥ 840px) -->
  <nav class="m3-nav-drawer layout-drawer">
    <div class="drawer-header"><h2 class="drawer-headline">newtil App</h2></div>
    <div class="drawer-content">
      <a href="#" class="drawer-item drawer-item-active">
        <i class="m3-icon icon:home"></i>
        <span class="drawer-label">홈</span>
      </a>
      <a href="#" class="drawer-item">
        <i class="m3-icon icon:mail"></i>
        <span class="drawer-label">메일</span>
      </a>
    </div>
  </nav>

  <!-- Navigation Rail (tablet 600-839px) -->
  <nav class="m3-nav-rail layout-rail">
    <div class="rail-items">
      <a href="#" class="m3-deco icon:home deco-pos:top deco-active">홈</a>
      <a href="#" class="m3-deco icon:mail deco-pos:top">메일</a>
    </div>
  </nav>

  <!-- Main Area -->
  <div class="layout-main">
    <header class="m3-top-app-bar layout-header">
      <button class="bar-leading"><i class="m3-icon icon:menu"></i></button>
      <h1 class="bar-title">페이지 제목</h1>
    </header>
    <main class="layout-content">
      <!-- page content here -->
    </main>
  </div>

  <!-- Navigation Bar (mobile < 600px) -->
  <nav class="m3-nav-bar layout-bar">
    <a href="#" class="m3-deco icon:home deco-pos:top deco-active">홈</a>
    <a href="#" class="m3-deco icon:mail deco-pos:top">메일</a>
  </nav>

</div>
