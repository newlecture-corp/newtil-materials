m3-checkbox 테스트

1. 기본 체크박스

<label class="m3-checkbox">
  <input type="checkbox"> 동의합니다
</label>
<label class="m3-checkbox">
  <input type="checkbox" checked> 선택됨
</label>

2. 색상 변형

<label class="m3-checkbox checkbox-color:danger">
  <input type="checkbox" checked> danger
</label>

3. Indeterminate (부분 선택)

<label class="m3-checkbox checkbox-mode:indeterminate">
  <input type="checkbox" checked> 부분 선택
</label>

4. 고정 상태 (static)

<label class="m3-checkbox checkbox-state:hovered">
  <input type="checkbox"> hovered
</label>

5. 변수 오버라이드

<label class="m3-checkbox" style="--checkbox-check-background: #e91e63;
     --checkbox-state-color: #e91e63;">
  <input type="checkbox" checked> 커스텀 Pink
</label>

m3-dialog 테스트

12. 기본 다이얼로그

버튼을 클릭하면 다이얼로그가 열립니다.

알림

변경사항이 저장되지 않았습니다. 정말 나가시겠습니까?

<div class="m3-dialog dialog-open">
  <h2 class="dialog-headline">알림</h2>
  <p class="dialog-content">내용</p>
  <div class="dialog-actions">
    <button class="m3-btn btn:text">취소</button>
    <button class="m3-btn">확인</button>
  </div>
</div>

13. 아이콘 다이얼로그

정말 삭제하시겠습니까?

이 작업은 되돌릴 수 없습니다. 모든 데이터가 영구적으로 삭제됩니다.

14. 변수 오버라이드 다이얼로그

커스텀 스타일

--dialog-* 변수를 변경해서 완전히 다른 다이얼로그를 만들 수 있습니다.

<div class="m3-dialog" style="--dialog-border-radius: 0.5rem;
     --dialog-background: var(--color-primary);
     --dialog-headline-color: var(--color-on-primary);">
  ...
</div>

15. 다크모드

상단 Light/Dark 전환 후 다이얼로그 열어보세요. 모든 컴포넌트가 자동 전환.

다크모드 테스트
