m3-card 테스트

1. 카드 타입 (elevated / filled / outlined)

Elevated (기본)

그림자가 있는 기본 카드. M3 surface-container-low.

Filled

배경이 진한 카드. 그림자 없음. M3 surface-container-highest.

Outlined

테두리만 있는 카드. 그림자 없음. M3 surface.

<div class="m3-card">Elevated (기본)</div>
<div class="m3-card card:filled">Filled</div>
<div class="m3-card card:outlined">Outlined</div>

2. Hover 상태

마우스를 올리면 그림자가 깊어지고 state layer overlay가 적용됩니다.

Elevated hover

hover 시 level1 → level2

Filled hover

hover 시 level0 → level1

Outlined hover

hover 시 그림자 추가

4. 카드 내부 영역 (header / content / actions / media)

미디어 카드

상단에 이미지/비디오 영역이 있는 카드. card-media 클래스를 사용합니다.

일반 카드

header, content, actions 영역을 가진 표준 카드 레이아웃입니다.

<div class="m3-card">
  <div class="card-media">
    <img src="..." alt="...">
  </div>
  <div class="card-header">
    <h3>제목</h3>
  </div>
  <div class="card-content">
    <p>내용</p>
  </div>
  <div class="card-actions">
    <button class="m3-btn btn:text btn-size:xs">자세히</button>
    <button class="m3-btn btn-size:xs">확인</button>
  </div>
</div>

5. 변수 오버라이드 (커스텀 카드)

--card-* 변수를 inline style로 재정의해서 커스텀 카드를 만들 수 있습니다.

기본 카드

기본 변수 그대로

둥근 + 강한 그림자

border-radius + shadow 변경

브랜드 카드

배경+텍스트 색 변경

<div class="m3-card" style="--card-border-radius: var(--radius-full);
     --card-box-shadow: var(--shadow-xl);">
  둥근 + 강한 그림자
</div>

<div class="m3-card" style="--card-background: var(--color-primary);
     --card-color: var(--color-on-primary);">
  브랜드 카드
</div>
