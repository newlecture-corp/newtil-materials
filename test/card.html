m3-card 테스트

1. 카드 타입 (elevated / filled / outlined)

Elevated

기본 카드 · 그림자 있음

surface-container-low 배경에 level1 그림자. hover 시 level2로 상승.

Filled

배경이 진한 카드

surface-container-highest 배경. 그림자 없음. hover 시 level1 그림자 추가.

Outlined

테두리만 있는 카드

surface 배경에 outline-variant 테두리. 그림자 없음.
<div class="m3-card">
  <div class="card-media"><img src="..." alt="..."></div>
  <div class="card-header">
    <div class="card-titles">
      <h3 class="card-headline">제목</h3>
      <p class="card-subhead">부제목</p>
    </div>
  </div>
  <div class="card-content">본문</div>
  <div class="card-actions">
    <button class="m3-btn btn:text btn-size:xs">자세히</button>
    <button class="m3-btn btn-size:xs">확인</button>
  </div>
</div>

3. 가로 레이아웃 (card-layout:horizontal)

가로 카드

이미지가 왼쪽에 배치

왼쪽에 이미지, 오른쪽에 콘텐츠가 배치되는 가로 레이아웃 카드입니다.
<div class="m3-card card-layout:horizontal card:outlined">
  <div class="card-media"><img src="..."></div>
  <div>
    <div class="card-header">...</div>
    <div class="card-content">...</div>
    <div class="card-actions">...</div>
  </div>
</div>

4. 크기 변형 (compact / default / large)

Compact

작은 패딩, 작은 제목

콤팩트한 카드입니다.

Default

기본 크기

기본 크기 카드입니다.

Large

넓은 패딩, 큰 제목

큰 사이즈 카드입니다.
<div class="m3-card card-size:compact">...</div>
<div class="m3-card">...</div> <!-- default -->
<div class="m3-card card-size:large">...</div>

5. 변수 오버라이드

--card-* 변수를 변경해서 완전히 다른 카드 만들기

둥근 모서리

--card-border-radius: 1.5rem

브랜드 카드

배경색, 텍스트색, 라운딩 모두 변수로 변경

그라디언트

background-color 대신 linear-gradient 적용

<div class="m3-card" style="--card-border-radius: 1.5rem;
     --card-box-shadow: var(--shadow-xl);">...</div>

<div class="m3-card" style="--card-background: var(--color-primary);
     --card-color: var(--color-on-primary); --card-border-radius: 0;">...</div>
